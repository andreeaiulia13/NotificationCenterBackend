# Notification Center Backend

  * Ruby version: 2.2.5
  * Rails version: 5.2.7
  * Database: Postgresql12

Gems used:
  * devise
  * doorkeeper
  * apipie-rails
  * rspec-rails

The API documentation can be found at this link: http://localhost:3000/apipie after you run the server.

Run rails db:seed in order to populate your database.
